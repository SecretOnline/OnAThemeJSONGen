On A Theme JSON Generator

Making texture variations in Minecraft is annoying. It's a lot of tedious work, which includes copying and pasting files, only to edit a couple of things, then repeat it.

I made this to make that process a little easier. The JSON generator handles a lot of this for you, with plans to be able to do a most of the process for you.

Usage

In the block menu, click "New Block". A dialogue will pop up, and ask to fill in any information.

  • Block Name: This is the name of the block (i.e. allium).
  • Texture Name: The base name of the texture to use (e.g. flower_allium)
  • Number of Variants: The number of variations, not including the original.
  • Model Type:
    • Don't Add Model: Create the blockstates file, without generating the model files.
    • Full Block: A normal block. Uses cube_all as the base model.
    • Cross: For things like flowers and cobwebs.
    • Double Cross: For two block tall flowers.
    • Double Cross Tint: Same as "Double Cross", but changes based on biome.

Once you have created the block, you can change the weights of the models. In the listboxes, navigate to the model you want to weight, and put in the number. Higher numbers mean a higher chance of being selected.

When you've made and weighted the variants, go to the File menu and click "Create Unzipped Version".

This will create a folder called "OaT", and inside will be the files, already laid out for easy integration into a resource pack.
